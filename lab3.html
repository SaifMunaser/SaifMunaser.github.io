Lab 3: Bootstrap Layout

A simple page using a navbar, grid, list, and an image.

This page was built for Lab 3 and the main goal was to practice layout using Bootstrap. Instead of throwing everything in one spot, the content is organized so it’s easier to read and looks clean on different screen sizes.

The list and the image are placed side-by-side in the same grid row. When the screen gets smaller, they stack automatically, which makes the page feel more responsive without needing a lot of extra code.

What this page includes

  • A navbar with a link back to the homepage
  • A jumbotron with a title and short subheader
  • Text content that explains the purpose of the page
  • A rounded image next to this list using the grid system
